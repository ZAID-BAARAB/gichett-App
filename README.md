## Get started
1. clone repo
   ```bash
   git clone git@github.com:ZAID-BAARAB/gichett-App.git
   ```
2. 
   ```bash
      cd gichett-App
   ```
3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
   npx expo start
   ```
